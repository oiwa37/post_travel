# 旅の思い出
旅行の思い出を投稿できるSNSアプリです。

* 自分の記事を投稿し、シェアする。

* 他のユーザの投稿を閲覧する。

* 日本地図には色塗り機能があり、訪れた場所に色をつけたり、行ってみたい場所に色付けをすることができる。
